Mark Brown – Personal Website
Overview

This repository hosts my personal website, designed to showcase my experience, skills, projects, and professional background in cybersecurity, IT, and GRC. The website serves as an online portfolio and central hub for my resume, GitHub projects, certifications, and contact information.

*Features

Professional Introduction: Includes a brief bio and career aspirations

Resume & Certifications: View and download my resume and CompTIA A+ certification

Projects Portfolio: Highlights my Python and cybersecurity projects with links to GitHub repositories

Contact Links: Easily accessible LinkedIn, GitHub, and email links for recruiters or collaborators

Clean Design: Modern, responsive layout with a dark-blue theme and professional typography


*Built With

HTML5 – Structure and semantic content

CSS3 – Styling, colors, fonts, and responsive layout

GitHub Pages – Hosting the live website

Optional JavaScript – For interactivity and dynamic elements


*Goals & Learning Outcomes

Build a professional online presence as a cybersecurity and IT student

Demonstrate proficiency in web design and GitHub Pages deployment

Showcase projects, skills, and certifications in a recruiter-friendly format

Gain experience with responsive design and portfolio presentation


*What I Learned

Through building this personal website, I developed skills in:

Structuring content for professional and recruiter-friendly presentation

Styling web pages using CSS for layout, typography, and colors

Deploying and maintaining a live website on GitHub Pages

Linking and organizing projects, resumes, and certifications effectively

Thinking about user experience and accessibility for professional portfolios
